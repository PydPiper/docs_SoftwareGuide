Acknowledgements: Content made better by K. Kim!

Abstract¶

This guide is a collection of references and examples for common commands/examples mainly structured around python, shell (unix/linux/cmd), excel related vba and other similar frameworks. All documentation are open source, and can be found on public domain.

The objective is to layout concepts and examples step by step in a clear and concise manner with emphasis on the “gotchas”.

If you enjoy what you find here or have suggestions for improvement, feel free to drop a comment on the github page: https://github.com/PydPiper/viks_SoftwareGuide
